# sertifikat
Proyek ini bertujuan untuk membuat halaman web verifikasi sertifikat digital (verification page) yang dapat diakses dengan memindai QR code pada sertifikat fisik/digital, atau dengan memasukkan ID sertifikat secara manual. Halaman ini akan dirancang dengan estetika premium, modern, dan responsif (dioptimalkan untuk perangkat mobile)
